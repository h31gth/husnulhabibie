About Us

Hai, perkenalkan nama saya Husnul Habibie dan saya adalah seorang lulusan teknik informatika di Universitas Muhammadiyah Riau.

Ini adalah portofolio saya, berisikan tentang saya dan kegiatan saya. Sebagai lulusan S1 Teknik Informatika dari Universitas Muhammadiyah Riau, saya memiliki pengalaman dalam web programming dan kesukaan dalam editing foto dan video. Kombinasi keterampilan teknis dan kreatifitas visual memungkinkan saya untuk menciptakan karya inovatif. Saya siap menghadapi tantangan baru dalam karier di bidang teknologi.
